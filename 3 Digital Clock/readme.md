# Digital Clock

This program is a simple digital clock implemented in C. It displays the current time and date in either 12-hour or 24-hour format, based on user preference.

## Features:

- **Real-time Display:** Continuously updates and displays the current time and date.
- **User-friendly Interface:** Allows users to choose between 12-hour and 24-hour time formats.
- **Clear Screen:** Utilizes the system command to clear the screen for a clean display.

## How to Use:

1. **Choose Time Format:**
   - Upon running the program, users are prompted to choose between 12-hour and 24-hour time formats.
   - Enter `1` for 24-hour format or `2` for 12-hour format (default).
   
2. **View Time and Date:**
   - The program will continuously display the current time and date in the chosen format.
   - Time is updated every second.

3. **Exit Program:**
   - To exit the program, simply terminate the execution.

## How to Run:

1. **Compile:**
   - Compile the program using a C compiler such as GCC or Clang.

2. **Run:**
   - Run the compiled executable file to start the digital clock.

3. **Choose Format:**
   - Select the desired time format when prompted.

4. **View Time:**
   - The current time and date will be displayed and updated in real-time.
